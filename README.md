# Recolor
A Python packages to recoloring default string

### Install
```
pip install recolor
```

# Examples
- Fore
```py
import recolor

print(recolor.fore("Hello!", "green", reset=True))
```

- Backgrounds
```py
import recolor

print(recolor.bg("Danger", "red", reset=True))
```
